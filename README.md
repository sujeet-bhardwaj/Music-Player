# 🎵 Music Player

A responsive and interactive Music Player built using HTML, CSS, and JavaScript. This project allows users to play, pause, skip songs, and manage their music experience through a clean and user-friendly interface.

## 🚀 Features

- Play and Pause Music
- Next and Previous Track Navigation
- Responsive User Interface
- Dynamic Song Information Display
- Volume Control
- Local Storage Support for saving user preferences
- Smooth and Interactive UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 💾 Local Storage

This project uses JavaScript Local Storage to store user preferences and maintain data even after refreshing or reopening the browser.
